Структура документов:
<Наименование>	<Количество фаз>	<Количество вариантов>	<Размеры изображения>

Наименования файлов:
<Наименование>_p<Фаза>_v<Вариант>

Нумирация фаз и вариантов начинается с 0.
Маски для чисел не применяются (как есть).
Количество фаз для каждого варианта одинаков, это необходимдля возможности плавного скольжения между вариантами.
Размеры спрайтов должны быть кратны 64 и не обльше 256.
Используется для юнитов 35-градусная проекция.



Пример имен файлов:
RobotJax_p0_v0.png
RobotJax_p1_v0.png
...
RobotJax_p17_v3.png
